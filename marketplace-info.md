# Dedalus Marketplace Info

## Short Description
Parse PDFs into text, sections, tables, and references

## Long Description
Extract text from PDF URLs, identify document sections and headings, parse tables to structured data, and extract academic references. Best-effort extraction for digital PDFs. Tools: pdf_to_text, extract_sections, extract_tables, extract_references.
